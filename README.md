# Stok Kontrol Sistemi

YBS316 Nesne Yönelimli Programlama Dönem Projesi

## Öğrenci Bilgileri
- **Öğrenci No:** 20220305065
- **Üniversite:** Piri Reis Üniversitesi

## Proje Hakkında
Bu proje, işletmelerin ürün stoklarını takip edebileceği bir masaüstü uygulamasıdır.

## Özellikler
- Ürün ekleme, düzenleme, silme
- Stok giriş/çıkış işlemleri
- Ürün arama ve filtreleme
- Kategori yönetimi

## Kullanılan Teknolojiler
- Java
- Swing (GUI)
- Dosya tabanlı veri saklama

## OOP Konuları
- Inheritance
- Interface
- Polymorphism
- Generic Class/Method
- Generic Collections (List, Map)
- Lambda Functions
